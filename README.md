## üéπ What is it?
The **TR-DZ** is an open-source, fully programmable drum machine designed specifically for **Tech House & Minimal Techno**. 
Unlike generic sample players, it runs a custom **Hybrid DSP Engine** on the **Electro-Smith Daisy Seed (STM32)**, offering zero-latency performance and deterministic groove control.

## üöÄ Key Features (V19 Firmware)
* **Hybrid Kick Engine:** Sine synthesis + Filtered Click + Pitch Envelope.
* **Rumble Bass:** FM-modulated Sub-bass with deterministic Sidechain (0ms latency).
* **Full Drum Kit:** Bandpass Clap + High-Pass Open/Closed Hats.
* **The "Tech Factor":** A chaos knob that introduces swing and FM modulation on the fly.
* **16-Step Sequencer:** 5 real-time selectable patterns (Techno, House, Syncopated, Roll, Break).
